# Catan Connect For Japanese - Chrome拡張

Catan Connect用の日本語音声(+効果音)機能を追加するChrome拡張です。

## 機能

- 太陽のターン・・・ニワトリの鳴き声
- 月のターン・・・オオカミの鳴き声
- ダイス番号の日本語読み上げ

## インストール方法

1. Chrome を開いて `chrome://extensions/` にアクセス
2. 右上の「デベロッパーモード」をON
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このディレクトリを選択

## 使用方法

1. https://www.catan.de/modules/custom/catan_connect/ にアクセス
2. 言語は英語を選択
3. New Game を選択
4. 時間を選んで再生マークを選択
5. 左下にスピーカーと人の音声のマークが表示されたら、人の音声マークを選択して消音する（英語と重複するため）
6. 再生マークを選択してゲームを開始する
